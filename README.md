# InspectFlow AI

AI-Powered QAQC Agent for EPC & Solar Projects

## Overview

InspectFlow AI is an intelligent QAQC assistant built using Gemini and Google Cloud Agent Builder.

The system helps:

- Manage RFIs
- Manage NCRs
- Inspection tracking
- Quality reporting
- Workflow automation
- Compliance monitoring

## Technologies

- Google Cloud Agent Builder
- Gemini AI
- MongoDB
- React
- Vercel

## Features

### AI Inspection Analysis

Automatically reviews inspection findings.

### NCR Management

Suggests corrective actions.

### RFI Tracking

Tracks inspection requests.

### Smart Reporting

Generates QAQC reports.
